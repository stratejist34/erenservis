# Gemini 3.1 Pro — Eren Servis Site Analiz Raporu
**Tarih:** 2026-04-22
**Model:** Gemini 3.1 Pro
**Kapsam:** Next.js App Router + React 19 + Tailwind v4 tam kod taraması

---

## 1. Mantık Hataları (Logic Errors)

### G1. Veri Tekrarı (Data Duplication)
**Dosya:** `app/hizmetler/page.tsx`
**Sorun:** `DSG_FAULTS`, `AISIN_FAULTS` vb. statik arrays tanımlı. Ancak `lib/sanziman.ts` içinde `transmissions.commonFaults` zaten aynı veriyi zengin haliyle tutuyor. Tek-kaynak-doğru prensibi kırılıyor; bir yerde güncelleme diğerine yansımaz.
**Öneri:** `hizmetler/page.tsx` arrays'lerini sil, döngüyü `transmissions`'tan besle.

### G2. Dropdown Blur relatedTarget Riski
**Dosya:** `components/Header.tsx`
**Sorun:** "Araçlar" dropdown'da `onBlur` + `e.relatedTarget` kullanılıyor. React synthetic event'te veya hibrit cihazlarda `relatedTarget` `null` dönebilir → menü istemsiz kapanır.
**Öneri:** `useRef` + global `mousedown` listener ile "click outside" pattern'i uygula.

## 2. Kod ve Performans

### G3. **Aşırı Font Yüklemesi (KRİTİK PERF)**
**Dosya:** `app/layout.tsx`
**Sorun:** 7 font ailesi (Archivo, IBM_Plex_Sans, Geist, Geist_Mono, Saira_Semi_Condensed, JetBrains_Mono, Fraunces). LCP'yi ciddi düşürür, FOUT/FOIT yaratır.
**Öneri:** Max 3 font (Saira display, Geist body, JetBrains mono).

### G4. Static Generation + Suspense Çakışması
**Dosya:** `app/page.tsx`
**Sorun:** `force-static` + Suspense + `SymptomProvider` içindeki olası `useSearchParams` — build'de de-optimization riski.
**Doğrulama:** Build log'unda warning var mı kontrol et.

### G5. trailingSlash Redirect Zinciri
**Dosya:** `components/Header.tsx`
**Sorun:** Linkler slash'lı (`/hizmetler/`). Eğer `next.config.ts`'de `trailingSlash: true` yoksa 308 zinciri oluşur.
**Not:** `trailingSlash: true` zaten aktif → risk düşük.

## 3. Tespit Edilen Eksikler

### G6. Aktif Link Belirteci (Active State)
**Dosya:** `components/Header.tsx`
**Sorun:** Kullanıcı hangi sayfada — görsel işaret yok.
**Öneri:** `usePathname()` + `text-brass` vurgusu.

### G7. App Router Native Sitemap/Robots
**Sorun:** `next-sitemap.config.js` + elle `public/robots.txt` kullanılıyor. Modern App Router'da `app/sitemap.ts` + `app/robots.ts` standart.
**Öneri:** Native dinamik sitemap'e geçiş.

### G8. Breadcrumbs UI Eksikliği
**Sorun:** Schema var ama alt sayfalarda görsel breadcrumb nav eksik olabilir (envanter gerekli).

### G9. Görsel Optimizasyonu
**Dosya:** `components/Header.tsx` (logo)
**Sorun:** `next/image` `sizes` prop optimum değil.
