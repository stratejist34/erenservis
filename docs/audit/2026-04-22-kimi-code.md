## 🔴 Kritik Kod & Mantık Hataları

### K1. `contentDispositionType: 'attachment'` Resimleri İndirtiyor
**Dosya:** `next.config.ts` satır 17
**Sorun:** Kullanıcı resme tıklayınca tarayıcı indirme başlatır. UX + Google Images görünürlüğüne zarar.
**Çözüm:** `'inline'` veya satırı kaldır.

### K2. `typescript: { ignoreBuildErrors: true }` Tip Güvenliği Kapalı
**Dosya:** `next.config.ts` satır 9
**Sorun:** OOM workaround'u ama tüm TS hataları sessiz geçiliyor. Local dev'de fark edilmiyor.
**Çözüm:** `NODE_OPTIONS=--max-old-space-size=...` ile OOM'u çöz, flag'i kaldır.

### K3. `app/rehber/page.tsx` Eksik — /rehber/ 404
**Sorun:** 21 alt sayfa var, hub yok. Breadcrumb'lar bozuk link veriyor.
**Çözüm:** Landing page oluştur.
**Durum:** ✅ ÖNCEKI COMMIT'TE KAPANDI (`d4ada3a`)

### K4. FloatingCTA `trailingSlash` Uyumsuzluğu
**Dosya:** `components/FloatingCTA.tsx` satır 9
```ts
const HIDDEN_PATHS = ['/', '/iletisim', '/hakkimizda'];
```
**Sorun:** `trailingSlash: true` → `usePathname()` `/iletisim/` döndürür. Array slash'sız → CTA yanlış yerde gösteriliyor.
**Çözüm:** `['/', '/iletisim/', '/hakkimizda/']`

### K5. `lib/sanziman.ts` — Yanlış Servis Linki
**Satır 457:** Renault EDC → `/hizmetler/dsg-sanziman-tamiri/` (YANLIŞ)
**Sorun:** Ford PowerShift → `/hizmetler/eat-sanziman-tamiri/` (bu da şüpheli).
**Çözüm:** Her şanzıman kendi doğru hizmet sayfasına yönlensin.

### K6. **Hizmetler Sayfasında "730 GÜN" Garanti — YASAL RİSK**
**Dosya:** `app/hizmetler/page.tsx` satır 116
```tsx
<div>730 <span>GÜN</span></div>
```
**Çelişki:**
- `lib/constants.ts`: `GARANTI_SURESI = '6 ay'`
- `FinalCTA.tsx`: "6 ay garanti"
- `Footer.tsx`: "6 ay işçilik garantisi"
**YASAL:** CLAUDE.md kuralı: "Garanti yalnızca 6 ay — hukuki yükümlülük yaratır."
**ACİL:** "730 GÜN" ibaresini sil.

### K7. Non-Null Assertion Runtime Crash Riski
**Dosyalar:**
- `components/sections/HeroSectionDC.tsx:93`
- `components/sections/SymptomSelector.tsx:11`
- `components/sections/BrandSection.tsx:32`
- `components/sections/FinalCTA.tsx:21`

```ts
SYMPTOMS.find((s) => s.id === selectedId)!
```
**Sorun:** URL manipülasyonu / race condition → crash.
**Çözüm:** `?? SYMPTOMS[0]` fallback.
